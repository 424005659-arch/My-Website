UNDRAFTED - Responsive Portfolio Website

Project Description
- UNDRAFTED is a multi-page responsive website that represents a student’s journey in learning, growth, and improvement in life and web development.  
The theme is inspired by basketball, where being "undrafted" means you are still developing your skills, working hard, and preparing for your future success.

This project showcases semantic HTML5 structure, external CSS styling, and responsive design using Flexbox and media queries.

Website Pages

Home Page (index.html)
- Introduction to the UNDRAFTED theme
- Welcome section
- Basketball inspirations (Kobe Bryant, Kyrie Irving, Michael Jordan)
- Card-based layout with images and descriptions

About Page (about.html)
- Personal biography of the student
- Explanation of the UNDRAFTED theme
- Motivation and purpose of the website
- Profile section of the developer

Contact Page (contact.html)
- Functional contact form (name, email, message)
- Contact information section
- Side-by-side layout (desktop view)
- Responsive stacked layout (mobile view)

Features
- Semantic HTML5 structure
- External CSS styling (no inline styles)
- Responsive design (mobile, tablet, desktop)
- Flexbox layout system
- Card-based UI design
- Image handling with proper alt attributes
- Styled forms with validation fields
- Media queries for responsiveness

Design Theme - 
The website uses a basketball-inspired theme called "UNDRAFTED":
- Represents growth and improvement
- Orange (#ff7a00) and dark background (#111111)
- Sports-inspired card layout
- Clean and modern UI design

Technologies Used
- HTML5
- CSS3
- Flexbox
- Media Queries
- Responsive Web Design principles

Responsive Design -
The website is optimized for:
- Mobile devices (stacked layout)
- Tablets (adjusted spacing)
- Desktop (full layout with side-by-side contact section)

How to Run the Project:
1. Download or clone the repository
2. Open the folder
3. Open `index.html` in any browser
4. Navigate using the menu links
