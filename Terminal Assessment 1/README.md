UNDRAFTED - Responsive Website

Project Description
- UNDRAFTED is a multi-page responsive website that maps the education journey of a student and experiences of growth and improvement in their life and web development skills.
- The theme is based on basketball: to be "undrafted" means that you are still maturing your skill set, striving, and preparing for success in the future.
- This project demonstrates how the HTML5 structure is used to present semantic content, how external CSS styling can be applied, and how responsive design techniques (Flexbox and media queries) can be used.

Website Pages

Home Page (index.html)
- Introduction to the UNDRAFTED theme
- Welcome section
- Basketball inspirations (Kobe Bryant, Kyrie Irving, Michael Jordan)
- Card-based layout with images and descriptions

About Page (about.html)
- Personal biography of the student
- Explanation of the UNDRAFTED theme
- Motivation and purpose of the website
- Profile section of the developer

Contact Page (contact.html)
- Functional contact form (name, email, message)
- Contact information section
- Side-by-side layout (desktop view)
- Responsive stacked layout (mobile view)

Features
- Semantic HTML5 structure
- External CSS styling (no inline styles)
- Responsive design (mobile, tablet, desktop)
- Flexbox layout system
- Card-based UI design
- Image handling with proper alt attributes
- Styled forms with validation fields
- Media queries for responsiveness

Design Theme
The website uses a basketball-inspired theme called "UNDRAFTED."
- Shows progress and development
- Orange (#ff7a00) and dark background (#111111)
- Sports-inspired card layout
- Responsive API that adapts to a wide range of screen sizes.

Technologies Used
- HTML5
- CSS3
- Flexbox
- Media Queries
- RWD principles

Responsive Design
The website has been optimized for:
- Mobile devices in a stacked format.
- Tablets (adjusted spacing)
- Desktop (full layout with side-by-side contact details)

How to Run the Project
1. Download or clone the repository.
2. Open the folder.
3. Open `index.html` in any browser.
4. Navigate using the menu links.
